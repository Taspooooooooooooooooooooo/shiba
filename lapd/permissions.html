🛡 Permissions Reference

How access is structured in SHIBA PIMS: each rank gets a template of permissions, and officers can be given extra groups on top.

This page is for administrators only.

🎖 Rank Templates

The default permissions every officer of a rank receives.

🧩 Permission Groups

Named bundles an admin can assign to an officer from their Personnel File (🛡 Permissions tab).

⚖️ Policies

Central rules that combine a permission with conditions (rank, ownership, time). Every module checks these, so a rule change happens in one place.
